## Getting Started

1. Install [Node.js (v10+)](https://nodejs.org/en/download/)
1. clone https://github.com/EtherTW/crosslink.git
1. Install project dependencies: `npm install` or `yarn`
1. Run locally
   * `npm start` or `yarn start`
